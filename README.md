## Features

- Add tasks with title and description
- Drag and drop tasks across columns (To Do, In Progress, Done)
- Delete tasks
- Task count per column
- Persistent storage using browser localStorage
- Accessible modal behavior (overlay close, Escape close, focus handling)

## Project Structure

- `index.html` – App layout and modal markup
- `style.css` – Theme and layout styles
- `script.js` – Task logic, drag-and-drop, and persistence
- `screenshot/` – Place project screenshots here

## How to Run

1. Open `index.html` in your browser.
2. Click **Add new task** to create tasks.
3. Drag tasks between columns to update their status.
